Skip Links

  • Provide ways for users to skip to important sections of your website. This will help users using screenreaders navigate your site easier and more efficiently.
  • There is no real theming rule when it comes to styling your skip links, as such a lot of websites tend to hide them with class="visuallyhidden" with a special focus attribute so that sighted keyboard only users are able to see them.
  • It is important to see that the link points to a valid HTML ID, as often this is overlooked in implementation.
  • In this site we have used 'Jump to main content', but other valid examples of "skip" links include:
